## How to cite

If you use these data please cite
- the original source
  > Pache, Matthias (2023): Pech and the basic internal classification of Chibchan. International Journal of American Linguistics 89 (1): 81-103.
- the derived dataset using the DOI of the [particular released version](../../releases/) you were using
